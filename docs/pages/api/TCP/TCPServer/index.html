libchamomile.Net.TCPServer

TCP Server 관련 함수들이 모여있는 API 입니다.

void StartServer(int Port)

TCP 서버를 시작합니다.
Example
using system;
using libchamomile.Net;

public void foo(){
	int Port = 5555;

	// Client 접속 시 CallBack받는 이벤트를 연결합니다.
	TCPServer.OnClientConnectedEvent += TCPServer_OnClientConnectedEvent;

	// Client로부터 메세지 수신 시 CallBack받는 이벤트를 연결합니다.
	TCPServer.OnReceiveMessageEvent += TCPServer_OnReceiveMessageEvent;

	// 서버를 시작합니다.
	TCPServer.StartServer(Port);
}

	// OnClientConnectedEvent CallBack 이벤트 함수입니다.
	private void TCPServer_OnClientConnectedEvent(TcpClient Socket, string IP) {
	Console.WriteLine($"Connected IP : {IP}");
}

	// OnReceiveMessageEvent CallBack 이벤트 함수입니다.
	private void TCPServer_OnReceiveMessageEvent(string Message) {
	Console.WriteLine($"Received : {Message}");
}
Parameters
Parameters Comment
Int Port 서버를 시작할 Port를 입력합니다.
Returns 리턴값이 없습니다.
다만 에러 발생시 SocketException을 Throw합니다.

void SendMessage(TcpClient Socket, string Message)

서버에 접속한 클라이언트에게 메세지를 보냅니다.
Example
using system;
using libchamomile.Net;

public void foo(){
    int Port = 5555;

	// 서버를 시작합니다.
	TCPServer.StartServer(Port);

	// 클라이언트에게 메세지를 송신합니다.
	TCPServer.SendMessage(ClientSocket, "Test Message");			
}
Parameters
Parameters Comment
TcpClient Socket 클라이언트 소켓을 입력합니다.
String Message 보낼 메세지를 입력합니다.
Returns 리턴값이 없습니다.
다만 에러 발생시 SocketException을 Throw합니다.

void StopServer()

서버를 중단합니다.
Example

public void foo(){

	// 서버를 시작합니다.

	// 서버를 중단합니다.
	TCPServer.StopServer();
}
