Chamomile Library Page

libchamomile.Crpytography

암호화를 지원하는 함수들이 모여있는 API입니다.
AES, DES, RSA, SEED등의 암호화와 각종 Hash 출력을 지원합니다.

▶ AES Crypto

string EncryptAES(string Input, string Key, AESCryptoList List)

문자열을 AES로 암호화 합니다.
Example
using system;
using libchamomile.Cryptography;

public void foo(){
	string target = "Hello World";
	target = AESCrypto.EncryptAES(target, "KeyValue", AESCrypto.AESCryptoList.AES128);
	target = AESCrypto.EncryptAES(target, "KeyValue", AESCrypto.AESCryptoList.AES256);

	// 또는 확장 메소드 사용
	target = target.ToEncryptAES("KeyValue", AESCrypto.AESCryptoList.AES128);
	target = target.ToEncryptAES("KeyValue", AESCrypto.AESCryptoList.AES256);
}
Parameters
Parameters Comment
String Input 문자열을 입력합니다.
String Key 암호화 키를 입력합니다.
AESCryptoList List AESCryptoList 열거형에서 암호화 할 항목을 선택합니다.
Returns AES 암호화 된 문자열을 반환합니다.

string DecryptAES(string Input, string Key, AESCryptoList List)

AES로 암호화된 문자열을 복호화 합니다.
Example
using system;
using libchamomile.Cryptography;
			
public void foo(){
	string target1 = "Hello World".ToEncryptAES("KeyValue", AESCrypto.AESCryptoList.AES128);
	string target2 = "Hello World".ToEncryptAES("KeyValue", AESCrypto.AESCryptoList.AES256);

	target1 = AESCrypto.DecryptAES(target, "KeyValue", AESCrypto.AESCryptoList.AES128);
	target2 = AESCrypto.DecryptAES(target, "KeyValue", AESCrypto.AESCryptoList.AES256);

	// 또는 확장 메소드 사용
	target1 = target.ToDecryptAES("KeyValue", AESCrypto.AESCryptoList.AES128);
	target2 = target.ToDecryptAES("KeyValue", AESCrypto.AESCryptoList.AES256);
}
Parameters
Parameters Comment
String Input 암호화된 문자열을 입력합니다.
String Key 복호화 키를 입력합니다.
AESCryptoList List AESCryptoList 열거형에서 복호화 할 항목을 선택합니다.
Returns AES로 복호화 된 문자열을 반환합니다.

string EncryptRSA(string Input, string PublicKey)

문자열을 RSA로 암호화 합니다.
Example
using system;
using libchamomile.Cryptography;

public void foo(){
	/* 
	Private Key와 Public Key는 반드시 생성해주어야 하며
	가장 먼저 Private Key 먼저 생성해주어야 합니다.
	이미 Private Key와 Public Key의 대칭키 파일들을 가지고 있다면
	따로 생성하지 않고 파일로 불러들여도 됩니다.
	*/
	
	string PrivateKey = RSACrypto.CreateRSAPrivateKey();
	string PublicKey = RSACrypto.CreateRSAPublicKey();

	string target = "Hello World";

	string result = RSACrypto.EncryptRSA(target, PublicKey);

	// 또는 확장 메소드 사용
	target = target.toEncryptRSA(PublicKey);
}
Parameters
Parameters Comment
String Input 암호화할 문자열을 입력합니다.
String PublicKey Public Key를 입력합니다.
Returns RSA로 암호화된 문자를 반환합니다.

▶ Hash

string StringToHash(string Input, HashList hashList)

문자열을 Hash로 출력합니다.
[MD5, SHA1, SHA256, SHA384, SHA512]를 지원합니다.
Example
using system;
using libchamomile.Cryptography;

public void foo(){
	string target = "Hello World";

	Hash.StringToHash(target, Hash.HashList.MD5);
	Hash.StringToHash(target, Hash.HashList.SHA1);
	Hash.StringToHash(target, Hash.HashList.SHA256);
	Hash.StringToHash(target, Hash.HashList.SHA384);
	Hash.StringToHash(target, Hash.HashList.SHA512);

	// 또는 확장 메소드 사용
	// Hash.HashList에서 동일하게 사용 가능
	target = target.ToHash(Hash.HashList.MD5);
}
Parameters
Parameters Comment
String Input 암호 문자열을 입력합니다.
HashList List HashList 열거형에서 출력할 항목을 선택합니다.
Returns 문자열의 Hash 값을 출력합니다.
