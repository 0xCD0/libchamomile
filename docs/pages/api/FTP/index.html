libchamomile.Net.FTP

FTP 관련 함수들이 모여있는 API입니다.

void SetFTPAccount(string FTPHostIP, string ID, string Password)

FTP에 접속하기 위해 ID와 Password 정보를 기입하는 함수입니다. FTP Class 사용전에 반드시 호출하여 입력해야 합니다.
Example
using system;
using libchamomile.Net;

public void foo(){
	FTP.SetFTPAccount("192.168.0.1", "testAccount", "testPassword");
}
Parameters
Parameters Comment
String FTPHostIP FTP의 주소를 입력합니다.
192.168.0.1과 같은 형태로 입력합니다.
String ID FTP에서 사용되는 ID를 입력합니다.
String Password FTP에서 사용되는 Password를 입력합니다.
Returns 해당 함수는 Return값이 없습니다.

List<string> ListDirectory(string directory)

디렉토리의 리스트를 Generic으로 출력합니다.
Example
using system;
using libchamomile.Net;

public void foo(){
	// 이미 설정했을 경우 제외
	FTP.SetFTPAccount("192.168.0.1", "testAccount", "testPassword");

	List<string> DirectoryList = FTP.ListDirectory("/");
}
Parameters
Parameters Comment
String directory 디렉토리 이름을 입력합니다.
Returns 디렉토리 List<string>을 반환합니다.

bool Upload(string localFilename, string targetFilename)

FTP에 파일을 업로드 합니다.
Example
using system;
using libchamomile.Net;

public void foo(){
	// 이미 설정했을 경우 제외
	FTP.SetFTPAccount("192.168.0.1", "testAccount", "testPassword");

	bool result = FTP.Upload("c:\\test.txt", "/test/test.txt");
}
Parameters
Parameters Comment
String localFilename 로컬 파일의 경로를 입력합니다.
String targetFilename 파일을 저장할 FTP 경로를 입력합니다. 파일 이름까지 같이 입력하여야 합니다.
Returns 성공 시 true를, 실패시 false를 반환합니다.

bool Download(string sourceFilename, string localFilename, bool PermitOverwrite)

FTP에서 파일을 다운로드합니다.
Example
using system;
using libchamomile.Net;

public void foo(){
	// 이미 설정했을 경우 제외
	FTP.SetFTPAccount("192.168.0.1", "testAccount", "testPassword");

	bool result = FTP.Download("/test/test.txt", "c:\\test.txt", true);
}
Parameters
Parameters Comment
String sourceFilename FTP에서 전송받을 파일 경로를 입력합니다.
String localFilename 로컬 영역에 저장될 경로를 입력합니다.
Bool PermitOverwrite 만약 중복된 파일이 존재할 경우 덮어씌우기를 할지 bool 형태로 입력합니다.
True를 입력하면 중복 파일에 덮어씁니다.
Returns 성공 시 true를, 실패 시 false를 반환합니다.
