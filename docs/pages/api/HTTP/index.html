libchamomile.Net.HTTP

HTTP Request 함수들이 모여있는 API입니다.

string PostRequest(string url, string PostPrameter = "")

HTTP Post를 요청합니다.
Example
using system;
using libchamomile.Net;

public void foo(){
	string target = HTTP.PostRequest("https://postman-echo.com/post", "test=test12&test2=test1234");

	//또는
	target = HTTP.PostRequest("https://postman-echo.com/post?test=test12&test2=test1234");
}
Parameters
Parameters Comment
String url Post할 주소를 입력합니다.
ex) http://192.168.0.1/restapi
String PostPrameter Post 데이터를 입력합니다.
ex) id=testAccount&password=testPassword
Returns 성공 시 Resonse Data를 리턴, 실패 시 null을 리턴합니다.

string GetRequest(string url, string GetParameter = "")

HTTP Get을 요청합니다.
Example
using system;
using libchamomile.Net;

public void foo(){
	string target = HTTP.GetRequest("https://postman-echo.com/get?test=1234");

	// 또는
	target = HTTP.GetRequest("https://postman-echo.com/get", "test=1234&id=1234");
}
Parameters
Parameters Comment
String url Get을 요청할 주소를 입력합니다.
ex) http://192.168.0.1/restapi
String GetParameter Parameter 데이터를 입력합니다.
ex)uid=testuid
Returns 성공 시 Resonse Data를 리턴, 실패 시 null을 리턴합니다.
