libchamomile.Net.UDPSender

UDP Sender 관련 함수들이 모여있는 API 입니다.

void SetUDPConfig(int _PortNumber, string _IPBandWidth)

UDP Sender를 사용하기 전에 Port 번호와 IPBandWidth를 설정하는 함수입니다.
Example
using system;
using libchamomile.Net;

public void foo(){
	int Port = 5555;

	// UDPSender 속성을 설정합니다. IP Address의 마지막 255는 BroadCast 주소입니다.
	UDPSender.SetUDPConfig(Port, "192.168.0.255");
}
Parameters
Parameters Comment
Int _PortNumber Port 번호를 입력합니다.
String _IPBandWidth BroadCast 대역대로 메세지를 전송하려면
192.168.0.255 형태의 BoradCast 주소를 String으로 입력합니다.
Returns 리턴값이 없습니다.
다만 에러 발생시 SocketException을 Throw합니다.

void SendMessage(string message)

UDP 메세지를 전송합니다.
Example
using system;
using libchamomile.Net;

public void foo(){
	int Port = 5555;

	// UDPSender 속성을 설정합니다. IP Address의 마지막 255는 Broadcast 주소입니다.
	UDPSender.SetUDPConfig(Port, "192.168.0.255");

	// BroadCast 메세지를 전송합니다.
	UDPSender.SendMessage("Test Message");
}
Parameters
Parameters Comment
String Message 전송할 메세지를 입력합니다.
Returns 리턴값이 없습니다.
다만 에러 발생시 SocketException을 Throw합니다.
