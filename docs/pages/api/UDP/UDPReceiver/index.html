libchamomile.Net.UDPReceiver

UDP Receiver 관련 함수들이 모여있는 API 입니다.

void StartUDP(int _PortNumber)

UDP BroadCast 메세지 수신을 시작합니다.
Example
using system;
using libchamomile.Net;

public void foo(){
	int Port = 5555;

	// BroadCast 메세지 수신 시 호출되는 CallBack 이벤트 입니다.
	//*Unity Engine일 경우 사용하시지 마십시오*
	UDPReceiver.OnReceiveMessage += UDPReceiver_OnReceiveMessage;

	///////////////////// Unity에서 사용하실 경우 /////////////////////
	/*  
	Unity Engine에서는 Listen이나 Receive시 Sync Wait상태에 진입할 경우,
	별도의 Thread를 사용하더라도 엔진이 멈추게 되는 문제가 있습니다.
	두번째는 UI Thread(Unity Engine상의 Main Thread) 외의 Thread가 UI Object에 접근 시 오류가 발생합니다.
	이를 해결하기 위해 MessageQueue를 자체적으로 구현하여 메세지 수신 시 MessagqQueue에 Enqueue하여
	평시적으로 구동되고 있는 Update 함수에서 Dequeue하여 수신된 메세지들을 꺼내서 사용하는 방식을 사용하였습니다. 
	*/

	// isUnityEngine 항목을 반드시 true로 체크해주셔야 합니다.
	UDPReceiver.isUnityEngine = true;

	// UDP BroadCast 수신을 시작합니다.
	UDPReceiver.StartUDP(Port);

	// BroadCast 메세지를 수신했을 때 UDPReceiver.MessageQueue로 메세지가 Enqueue됩니다.
	// Update 함수에서 MessageQueue를 Dequeue하여 사용하십시오.
	string message = TCPClient.MessageQueue.Dequeue();
}

public void UDPReceiver_OnReceiveMessage(string message) {
	Console.WriteLine(message);
}
Parameters
Parameters Comment
Int _PortNumber Port 번호를 입력합니다.
Returns 리턴값이 없습니다.
다만 에러 발생시 SocketException을 Throw합니다.

void StopUDPReceiver()

UDP BroadCast 메세지 수신을 중단합니다.
Example
using system;
using libchamomile.Net;

public void foo(){
	int Port = 5555;

	// UDP BroadCast 수신을 시작합니다.
	UDPReceiver.StartUDP(Port);

	// UDP BroadCast 수신을 중단합니다.
	UDPReceiver.StopUDPReceiver();
}
Parameters
Parameters Comment
입력할 인자가 없습니다.
Returns 리턴값이 없습니다.
다만 에러 발생시 SocketException을 Throw합니다.
