Chamomile Library Page

libchamomile.Data

Data의 직렬화/역직렬화 또는 XML 출력/불러오기 등의 함수가 모여있는 API입니다.

bool SerializeDataToFile<T>(T targetClass, string fullFilePath)

대상 객체를 직렬화하여 File로 저장합니다. targetClass는 반드시 Header로 [Serialize()]를 기입하셔야 합니다.
Example
using system;
using libchamomile.Data;

[Serializable()]
public class fooClass {
	public string fooString;
	public int barInt;
	public float fooFloat;
	public double barDouble;
}

public void foo() {
	string filepath = $"{Environment.CurrentDirectory}//output.bin";

	fooClass foo = new fooClass();
	foo.fooString = "Foo String";
	foo.barInt = 1234;
	foo.fooFloat = 123.45f;
	foo.barDouble = 123.45f;

	bool result = Data.SerializeDataToFile(foo, filepath);

	// 또는 확장 메소드 사용
	bool result = foo.ToSerializeDataToFile(filepath);
}
Parameters
Parameters Comment
T targetClass 직렬화하여 저장할 Class를 입력합니다.
String fullFilePath 저장할 파일 경로를 입력합니다. 파일 이름 및 확장자까지 입력해야 합니다.
Returns 성공 시 true, 실패 시 exception을 throw 합니다.

string SerializeDataToJson<T>(T targetClass)

대상 객체를 직렬화하여 Json String으로 반환합니다. targetClass는 반드시 Header로 [Serialize()]를 기입하셔야 합니다.
Example
using system;
using libchamomile.Data;

[Serializable()]
public class fooClass {
	public string fooString;
	public int barInt;
	public float fooFloat;
	public double barDouble;
}

public void foo() {
	// Json 변환을 위해서는 Newtonsoft의 Json Nuget Package를 설치해야 합니다.
	
	fooClass foo = new fooClass();
	foo.fooString = "Foo String";
	foo.barInt = 1234;
	foo.fooFloat = 123.45f;
	foo.barDouble = 123.45f;

	string jsonData = Data.SerializeDataToJson(foo);

	// 또는 확장 메소드 사용
	jsonData = foo.ToSerializeDataToJson();
}
Parameters
Parameters Comment
T targetClass 직렬화하여 Json으로 출력할 Class를 입력합니다.
targetClass는 반드시 Header로 [Serialize()]를 기입하셔야 합니다.
Returns 성공 시 Json String을 반환, 실패 시 exception을 throw 합니다.
